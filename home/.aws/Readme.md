# home/.aws

Configuration for AWS tools.

At minimum, this directory will usually contain two files:

## `.aws/credentials`

```
[default]
aws_access_key_id = <access_key>
aws_secret_access_key = <secret_key>
```

## `.aws/config`

```
[default]
region = <region>
```

More information:

https://github.com/aws/aws-cli
